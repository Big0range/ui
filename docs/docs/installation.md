# 安装

## 环境支持

不支持 IE11 <br>
需预先安装element-plus<br>
支持typescript


## 使用包管理器

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 @big0range/ui, 然后您就可以使用打包工具，例如 Vite 和 webpack

```
# 选择一个你喜欢的包管理器

# NPM
$ npm install @big0range/ui

# Yarn
$ yarn add @big0range/ui

# pnpm
$ pnpm install @big0range/ui
```
